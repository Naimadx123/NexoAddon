# InventoryType

The Inventory Type allows you to open inventory with various types!

### Open anvil inventory

```yaml
# Example

Mechanics:
  inventoryType:
    type: ANVIL
    title: <red>HI!
```

### Working workbench with custom name

```yaml
# Example:

Mechanics:
  inventoryType:
    type: WORKBENCH
    title: <red>HI!
```
